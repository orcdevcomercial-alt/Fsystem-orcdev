💰 ORCDev Finance System

Sistema SaaS de gerenciamento financeiro desenvolvido para auxiliar pessoas físicas, profissionais autônomos e pequenas empresas no controle de receitas,
despesas e planejamento financeiro.

---

📋 Sobre o Projeto

O ORCDev Finance é uma plataforma web moderna criada para oferecer uma experiência simples, intuitiva e eficiente no gerenciamento financeiro.

O sistema permite registrar movimentações financeiras, acompanhar relatórios, visualizar indicadores de desempenho e manter maior controle sobre a saúde financeira do usuário.

Este projeto foi desenvolvido com foco em boas práticas de programação, usabilidade, segurança e organização de dados, aplicando conhecimentos adquiridos durante a formação técnica e complementar do desenvolvedor.

---

🚀 Funcionalidades

Dashboard Financeiro

- Visão geral das finanças
- Saldo atual consolidado
- Resumo de receitas
- Resumo de despesas
- Indicadores financeiros


Gestão de Receitas

- Cadastro de receitas
- Histórico de entradas
- Organização por categorias

Gestão de Despesas

- Cadastro de despesas
- Controle de gastos
- Classificação por categorias

Relatórios

- Análises financeiras
- Comparativos de movimentações
- Acompanhamento da evolução financeira
- Relatórios gerados em PDF, para maior controle e visibilidade de gastos e entradas

Autenticação de Usuários

- Cadastro de contas
- Login seguro
- Recuperação de acesso
- Gerenciamento de sessão

- Armazenamento seguro de credenciais
